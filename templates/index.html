About This Project

Delhi House Price Prediction is an intelligent web application built to estimate property prices in Delhi based on key features like BHK, bathrooms, parking, area, and furnishing status.

This project leverages machine learning models trained on real estate data to provide accurate, instant price predictions — helping buyers, sellers, and renters make informed decisions.
